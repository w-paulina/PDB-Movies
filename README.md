# PDB-Movies
Engineering project entilted "Video streaming platform with social media features". The important features of the application are social functions, that enable users to make friends, add comments about movies, rate movies, and moreover, to recommend them to other users. User has access to all movies he has rated, added to playlists, and to those recommended by his friends by the implementation of the "My List" module.
